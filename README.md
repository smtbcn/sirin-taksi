# Şirin Taksi Web Sitesi

Modern ve kullanıcı dostu taksi hizmet web sitesi.

## Özellikler

- 🚕 Responsive tasarım
- 📱 WhatsApp ile kolay iletişim
- ⭐ Müşteri yorumları sistemi
- 🔒 Admin paneli
- 📊 SEO optimizasyonu
- 🎯 Google Ads entegrasyonu

## Kurulum

1. Projeyi klonlayın:

```bash
git clone https://github.com/yourusername/sirin-taksi.git
```

2. Bağımlılıkları yükleyin:

```bash
cd sirin-taksi
npm install
```

3. `.env.example` dosyasını `.env` olarak kopyalayın ve gerekli değişkenleri ayarlayın.

4. Geliştirme sunucusunu başlatın:

```bash
npm run dev
```

## Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- FontAwesome
